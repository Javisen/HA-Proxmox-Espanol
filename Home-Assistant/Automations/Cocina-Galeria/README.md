Algunas automatizaciones para electrodomesticos o iluminacion, control de presencia, etc

- Hardware necesario: Sensores de movimiento, enchufes con medición de energía y sensores de puerta.
- Ayudantes (Helpers): input_boolean para marcar que hay ropa dentro de la lavadora.
- Integraciones: Zigbee2MQTT  para los sensores
